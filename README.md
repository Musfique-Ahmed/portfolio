# Musfique Ahmed Portfolio

Modern static portfolio focused on recruiter-first navigation and data-science case-study storytelling.

## Current structure

- index.html: Canonical homepage with impact metrics and featured work
- projects.html: Dedicated case-study portfolio with domain filters
- all-experiences.html: Grouped professional timeline with expandable role details
- all-awards.html: Awards, scholarship milestones, and publication spotlight
- all-blogs.html: Featured writing plus archived notes
- resume.html: Resume-style overview with print/save-to-PDF action
- contact.html: Dedicated contact page with Formspree integration
- update.html: Redirect to canonical homepage
- blog-post-1.html to blog-post-5.html: Individual article pages
- assets/css/main.css: Shared design tokens and component styles
- assets/js/main.js: Shared navigation, animation, filters, and form behavior
- sitemap.xml: XML sitemap for search indexing
- robots.txt: Crawl directives and sitemap pointer

## Design direction

- Dark analytics theme with clear contrast and data-grid atmosphere
- Recruiter-first information hierarchy
- Mobile-first responsive layout
- Shared UI components and navigation across all pages

## Local preview

Open index.html in a browser, or use any static server.

## Notes

The strongest blog posts are featured, while earlier tutorial notes are intentionally marked as archived for future rewrite.

All major pages now include canonical metadata, Open Graph and Twitter card tags, and JSON-LD structured data.
